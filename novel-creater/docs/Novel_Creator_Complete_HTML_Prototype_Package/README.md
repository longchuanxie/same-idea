# Novel Creator 完整 HTML 原型

本包包含一个完整的单文件 HTML 原型，已整合此前主流程页面，并补齐 Gap Closure 报告要求的页面和状态。

## 打开方式

直接用浏览器打开：

`Novel_Creator_Complete_HTML_Prototype.html`

## 页面范围

共 20 个页面/状态：

1. 项目库首页
2. 新建项目向导
3. 隐私与模型调用确认
4. 项目概览
5. 写作编辑器
6. 大纲与资料库
7. 联网调研与引用
8. 修订追踪与审核
9. Agent 任务中心
10. 待确认变更中心
11. 角色详情
12. 世界观设定
13. 笔记与素材库
14. 会话、分支与时光机
15. 导出向导
16. 数据与备份
17. 设置与服务商
18. 空状态与错误态
19. 移动端编辑器
20. 移动端 Agent 助手

## 设计约束

- 莫兰迪色系
- 无 emoji
- Heroicons 风格内联 SVG
- 本地优先
- Agent 写入必须可审核
