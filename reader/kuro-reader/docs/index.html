Kuro Reader

一款专注于本地阅读体验的跨平台应用。支持漫画与文本阅读，覆盖 Web 与 Android，提供沉浸式阅读与流畅交互动画。

功能特性

为阅读而生

从漫画到文本，从本地到云端，Kuro Reader 覆盖你所需的一切阅读功能。

多格式支持

支持漫画压缩格式 ZIP/CBZ/RAR/CBR，以及文本格式 TXT/EPUB，直接导入即可阅读。

双模式阅读

漫画支持条漫/页漫模式自由切换、双页模式；文本支持自定义字体与排版，阅读体验舒适。

云端连接

支持 WebDAV、NAS、FTP 等多种云端来源，随时随地同步你的书库。

隐私保护

手势密码锁保护你的私密内容，所有数据存储在本地，隐私完全由你掌控。

智能管理

子书库分类、标签系统、收藏功能，配合批量导入，轻松管理海量书籍。

流畅动画

Material Design 3 设计体系，全 CSS 动画系统带来丝滑的页面过渡与交互反馈。

安装指引

从零开始

无论你是普通用户还是开发者，都可以快速上手。

  • 1

    下载 APK（Android）

    直接点击上方「下载 APK」按钮，或前往 GitHub Releases 页面获取最新版本。

    https://github.com/longchuanxie/same-idea/releases
  • 2

    允许安装未知来源应用

    Android 系统会提示安全警告，前往 设置 → 安全 → 允许安装未知来源应用，对你使用的浏览器或文件管理器授权。

  • 3

    安装并打开

    点击 APK 文件进行安装，完成后打开 Kuro Reader 即可开始使用。

  • 4

    导入书籍

    通过应用内的导入功能，选择本地 ZIP/CBZ/RAR/CBR 漫画文件或 TXT/EPUB 文本文件，也可连接 WebDAV / FTP 云端来源批量导入。

  • 1

    克隆仓库

    确保已安装 Git，克隆项目到本地。

    git clone https://github.com/longchuanxie/same-idea.git
  • 2

    安装依赖

    需要 Node.js 18+，在项目根目录执行：

    npm install
  • 3

    Web 开发模式

    启动本地开发服务器，支持热更新：

    npm run dev
  • 4

    构建 Web 生产包

    编译优化后的生产版本，输出到 dist/ 目录：

    npm run build
  • 5

    构建 Android APK

    需要 Java 17+ 和 Android Studio。先同步 Web 资源，再编译 APK：

    npx cap sync android
    cd android && ./gradlew assembleRelease
  • 6

    获取 APK 文件

    构建产物位于 android/app/build/outputs/apk/release/app-release.apk，可传输到手机安装。
